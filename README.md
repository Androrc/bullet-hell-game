# 2D Bullet Hell Game

A 2D **Bullet Hell** game developed in **Unity** with Pixel Art graphics.  

This project was created by a **team of 4** as a final year high school TCC during the **Digital Game Programming** technical course. 

![Gameplay](assets/gameplay.gif)

The game also **subtly addresses social issues** such as:  
- Workplace abuse  
- Misallocation of roles  
- Burnout  

---

## 🎮 Gameplay

- Smooth player movement  
- Enemies with varied shooting patterns  
- Challenging boss fights  
- Health system and visual indicators  

> The game focuses on delivering a **challenging and engaging experience** while incorporating important social messages in a subtle way.

---

## 🛠️ Technologies

- **Engine:** Unity 2D  
- **Language:** C#  
- **Art:** Pixel Art  

---

## 📂 Project Structure

- `Assets/` – Scripts, sprites, prefabs, and scenes  
- `Library/` – Auto-generated Unity files (not tracked by Git)  
- `ProjectSettings/` – Project configuration files  

> Temporary and heavy files are ignored via `.gitignore`.

---

## 🔧 How to Run

1. Clone or download the repository  
2. Open the project in **Unity** (version 2020 or later recommended)  
3. Open the main scene located in `Assets/Scenes`  
4. Click **Play** to start the game
